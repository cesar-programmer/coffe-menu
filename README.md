# Koa Caffe Menu

Menu digital SPA para Koa Caffe construido con React, Vite y Tailwind CSS.

## Comandos

```bash
npm run dev
npm run build
npm run preview
```

## GitHub Pages

El proyecto usa `base: './'`, por lo que puede publicarse en GitHub Pages bajo `https://usuario.github.io/repositorio/` sin cambiar rutas de assets.

Para desplegar con `gh-pages`:

```bash
npm run deploy
```

## QR

Cuando tengas la URL final de GitHub Pages, genera el QR con:

```bash
npm run qr -- --url=https://usuario.github.io/repositorio/
```

El archivo se guarda en `public/koa-caffe-qr.png`.
